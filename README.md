## The Hand Lab

Embedded in the Phase I–IV study halls are **26 animated, interactive hand histories** from Adrián Mateos (2024 WSOP + Bahamas final table) and Leo Margets (2025 WSOP run to 7th) — real documented hands replayed on an animated table. The action pauses at every hero decision: pick your line (bet sizes included), get the verdict against what the pro actually did, then the live read and key lesson from the source. Matched-decision best scores save per hand. Phase III Day 55 hosts the **Aggression Gym** — eight attack-then-exit drills from the aggression-drills database — and Phase II Day 39 holds the full casebook index with links to every hand.

Where cards or boards weren't preserved in the sources, hands carry a **◆ reconstructed** badge: decisions, sizings and reasoning are verified; the exact cards are plausible reconstructions consistent with every described constraint.

## How to use

Open `wsop-100-day-plan.html` in any browser — everything is self-contained, no server needed. The study halls are linked from the nav bar and from each phase header. Each study day has a lesson and a drill (8 questions, options shuffled every attempt); every 7th day is a consolidation gauntlet, every 20th a phase exam. Pass at 80%+, mastery at 90%+. Progress and best scores save in your browser's localStorage.

## Notes

Ranges are simplified live-deep tournament defaults for a Main Event field, with combo counts where precision matters — learn the pattern, then adapt to the table. Not solver gospel, and not financial advice: bankroll and staking sections are frameworks, not recommendations.

# Claude Prompt — WSOP Main Event 100-Day Prep Program

Adapted from the "WSOP Main Event Preparation Report and Claude Prompt" (original was 30-day; this version reflects the customized 100-day / gladiator–Mad Max–zen monk build).

---

```
You are an elite tournament poker strategist and performance coach.
Your task is to design a rigorous 100-day daily poker training program tailored
to prepare a player for the next WSOP Main Event.

Persona and framing:
- The player's identity is one third gladiator, one third Mad Max, and one
  third zen monk. Map the three stylistic models onto that trinity:
  - Gladiator = Alex Foxen: theory-forged, fearless at the highest-leverage
    preflop nodes, "understand, don't memorize."
  - Mad Max = Shaun Deeb: freezeout-as-wasteland survival, grind endurance,
    believable range storytelling, shows up every day win or lose.
  - Zen Monk = Adrián Mateos: no fixed strategy, adaptive pressure, feel the
    emotion and still perform, river calls made because the range says so.
- Use this trinity consistently in the program's language and structure.

Important context and constraints:
- Player skill level: unspecified. Do NOT invent a level; explicitly note that
  it is unspecified and provide a modular plan that can scale for intermediate
  through advanced players.
- Bankroll: unspecified. Do NOT assume a bankroll; explicitly note that it is
  unspecified and give conservative bankroll and series-planning frameworks for
  both self-funded and partially sold/satellite-based approaches.
- Tournament target: next WSOP Main Event. If the exact structure for that year
  is not publicly available, say so and use the latest officially published WSOP
  Main Event schedule/structure as the best proxy, clearly labeling it as a proxy.
- You must explicitly compare the three players' tendencies, identify
  commonalities and differences, and then synthesize an aggressive-but-survivalist
  multi-day WSOP strategy from them.
- The result must be written in English and should read like a serious
  analytical report, not a casual blog post.

Research priorities:
- Prioritize primary and near-primary sources when available:
  - Official WSOP schedule/structure/rules
  - Published hand histories
  - Official interviews or direct player quotes
  - Official training site materials
  - Recent high-stakes tournament reports
- If web access is available, cite claims carefully with source attribution
  throughout.
- If web access is not available, still produce a complete plan, but clearly
  label where current official structure details could not be verified.

What the report must include:
- Begin with an executive summary.
- State assumptions and explicitly list what is unspecified.
- A comparative table of Mateos vs Foxen vs Deeb tendencies.
- A synthesized WSOP Main Event style profile derived from those three players.
- Explicit strategic sections covering:
  - Pre-flop ranges
  - Post-flop lines
  - Turn plans
  - River plans
  - ICM and bubble play
  - Deep-stack adjustments
  - Mid-stack adjustments
  - Short-stack adjustments
  - Blind/ante structure implications across multiple days
  - Table dynamics
  - Exploitative vs GTO balance
  - Multi-day stamina and tilt management
- A day-by-day WSOP multi-day strategy table showing the correct balance of
  survival vs aggression by tournament day.
- A complete 100-day day-by-day curriculum organized as five 20-day phases
  (baseline forging, advanced postflop, ICM/survival, live-field execution,
  integration and taper), with for every day:
  - Daily objective
  - Time allocation (default 2.5–4.5 hours, scalable down 40% or up 50%)
  - Drills
  - Study materials
  - Hand-history review work
  - Mental preparation
  - Physical preparation
  - Bankroll/session/logistics tasks where relevant
  - Daily measurable checkpoint
- Every 7th day is a consolidation day; every 20th day is a phase exam.
- An interactive-quiz question bank (multiple choice, with correct answers and
  explanations) covering preflop, postflop, ICM/bubble, and mental game,
  organized by the three thirds (gladiator / Mad Max / zen monk), designed to
  be retaken on Day 1, Day 50, and Day 90 to identify the weakest third.
- Templates/tables for:
  - Daily progress tracking
  - Weekly review
  - Bankroll and series planning
  - Tournament-day routine
  - Hand-history review
- At least 3 sample hand-history analyses, preferably tied to published hands or
  clearly labeled illustrative examples.
- A sample weekly review filled out as an example.
- Mermaid diagrams for:
  - Tournament-day flow
  - In-game decision tree
- Include tables wherever useful.

Style requirements:
- Be analytical, specific, and practical.
- Do not produce vague motivational language.
- Do not default to "play GTO." Explain when to use theory baselines and when to
  deviate exploitatively in a Main Event field.
- Emphasize that the WSOP Main Event is a freezeout marathon and explain how
  that changes aggression thresholds.
- Highlight where the strategy should differ if the player is covered, is the
  covering stack, or is one of the shortest stacks at the table.
- Use realistic metrics:
  - preflop quiz accuracy
  - ICM quiz accuracy
  - marked-hand count
  - review completion rate
  - sleep consistency
  - exercise completion
  - emotional control / tilt incidents
  - decision-quality self-grades
- Avoid overfitting to solver outputs; prioritize understanding, pattern
  recognition, opponent profiling, and repeatable in-game heuristics.

Deliverable structure:
- Executive Summary
- Assumptions and Structural Proxy
- Comparative Pro Tendencies Table
- Synthesized WSOP Strategy Blueprint (the trinity)
- 100-Day Daily Training Program (five phases)
- Interactive Quiz Question Bank
- Tracking Templates
- Sample Hand-History Analyses
- Sample Weekly Review
- Mermaid Diagrams
- Final Key Takeaways

Make the report detailed enough that a player could actually follow it for 100
days without needing major additions.
```
